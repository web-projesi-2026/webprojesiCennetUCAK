# Uçak Yazılım Endüstrileri

Modern, kurumsal ve endüstriyel odaklı bir yazılım şirketi için geliştirilmiş çok sayfalı portföy web sitesidir. Proje; güçlü bir kullanıcı deneyimi, mobil uyumluluk ve profesyonel arayüz tasarımı hedeflenerek hazırlanmıştır.

## Proje Amacı

Bu proje, bir yazılım ve endüstri şirketinin hizmetlerini tanıtmak, kurumsal kimliğini yansıtmak, ürün ve projelerini sergilemek ve kullanıcı ile iletişim kurmasını sağlamak amacıyla geliştirilmiştir.

## Site Yapısı

Proje toplamda 7 sayfadan oluşmaktadır:

- Anasayfa
- Hakkında
- Çalışan Kadrosu
- Projeler
- Ürünler
- Hizmetler
- İletişim

## Çalışan Kadrosu Özelliği

Sistemde 10 adet idari personel bulunmaktadır. Her çalışan için fotoğraf, pozisyon, eğitim bilgisi, deneyim ve özgeçmiş bilgileri yer almaktadır.

Kullanıcı deneyimini artırmak amacıyla çalışan detayları sayfa değiştirilmeden, kartlara tıklanarak modal (popup) yapı ile gösterilmektedir.

## Tasarım Özellikleri

Siyah, beyaz ve kırmızı ağırlıklı modern bir tasarım kullanılmıştır. Arayüzde kart tabanlı yapı tercih edilmiş, hover efektleri ve ışık (glow) geçişleri ile kullanıcı etkileşimi artırılmıştır. Tasarım genel olarak kurumsal ve endüstriyel bir görünüm sunmaktadır.

## Mobil Uyumluluk

Proje tamamen responsive olarak geliştirilmiştir. Mobil cihazlar için optimize edilmiş, hamburger menü yapısı eklenmiş ve görsellerin taşmasını engelleyecek düzenlemeler yapılmıştır. CSS media query kullanılarak farklı ekran boyutlarına uyum sağlanmıştır.

- JavaScript (Vanilla JS)

## GitHub Çalışma Süreci

Proje geliştirme sürecinde GitHub iş akışı uygulanmıştır. Öncelikle haftalık görev için bir issue oluşturulmuş, ardından bu issue için ayrı bir branch açılmıştır. Geliştirme süreci birden fazla commit ile ilerletilmiş, sonrasında pull request açılarak ana branch’e merge edilmiştir. Yapılan değişiklikler GitHub Pages üzerinden canlı yayına aktarılmıştır.
